CRITICAL: Preserve the 4-way join logic EXACTLY as-is from the source script:
1. Kallisto summary -> CREATE TABLE kallisto (filtered by sample_id)
2. VirNucPro reads -> CREATE TABLE vnp (all rows)
3. FULL OUTER JOIN kallisto + vnp ON READ_ID -> joined_kv
4. Kraken2 reads -> CREATE TABLE k2 (filtered by sample_id)
5. FULL OUTER JOIN joined_kv + k2 ON regexp_replace(READ_ID, '/[12]$', '') -> joined_all
6. geNomad virus summary -> CREATE TABLE genomad (from TSV)
7. LEFT JOIN joined_all + genomad ON VNP_CONTIG_ID -> result (stamped with sample_id)
8. COPY result TO output (FORMAT PARQUET, COMPRESSION ZSTD)

Do NOT simplify, restructure, or reorder the joins.
